# Mascot

## Mascot description

### Mascot concept

A friendly orange tabby cat who feels like an indie developer from an optimistic retro future. The character is curious, inventive, and approachable rather than “hackery” or intense. The coding theme should be present but understated: the cat is simply working at a laptop, with the scene and styling carrying the identity. The overall look should feel suitable for CLI/docs branding and a landing page hero, so it should be readable, iconic, and not overloaded with detail.

### Detailed prompt

```
Create a simple cartoon mascot of a friendly orange tabby cat programmer in a retro-futuristic 1960s/1970s sci-fi style. The cat is sitting at a laptop, posed like an indie developer building something creative, with a curious and inventive expression. The character should feel warm, clever, approachable, and slightly whimsical.

The cat has recognizable orange tabby fur with soft stripes, rounded cheeks, expressive eyes, and a clean, memorable silhouette. It is wearing a simple hoodie that helps give it a modern developer personality while still fitting the retro-future world. The hoodie should be visually simple and not dominate the design.

Use a cream, orange, and teal palette with soft shaded colors, not flat-only coloring. The shading should be gentle and illustrative, giving the mascot warmth and depth while keeping the drawing clean and simple. Avoid harsh realism. The overall style should be simple cartoon character design, polished enough for branding, stickers, docs, and a landing page hero.

The retro-futuristic influence should come from space-age rounded shapes, mid-century sci-fi design language, and a sense of optimistic vintage technology. The laptop can have subtle retro details such as softened corners, chunky keys, or a slightly vintage-futuristic silhouette, but it should still read clearly as a laptop. Programming cues should be subtle: perhaps a minimal code screen glow, a tiny terminal-like interface, or a few understated geometric UI elements, without cluttering the image.

Composition should prioritize the mascot as the main subject, with a clean readable silhouette and enough negative space to work well in branding. The mood is creative, cozy, inventive, and developer-friendly. The illustration should feel like a mascot for a CLI tool or technical product: charming, smart, and distinctive, without being overly childish.

Style references: mid-century retro-futurism, 60s/70s sci-fi illustration, simple branded cartoon mascot, soft shading, clean linework, warm tech aesthetic. High readability, iconic shape language, minimal background, no text, no watermark.
```

### Short reusable prompt

```
A simple cartoon mascot of a friendly orange tabby cat programmer in a 60s/70s retro-futuristic sci-fi style, sitting at a laptop, wearing a hoodie, with a curious and inventive expression. Use a cream, orange, and teal palette with soft shaded colors. Keep the coding theme subtle, with clean branding-friendly composition, rounded space-age shapes, warm mid-century retro-tech atmosphere, minimal background, high readability, suitable for CLI/docs branding and landing page hero.
```
